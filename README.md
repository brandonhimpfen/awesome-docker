## CLI Tools

- [Docker CLI](https://docs.docker.com/engine/reference/commandline/cli/) – Core CLI for managing containers and images.
- [Docker Compose CLI](https://docs.docker.com/compose/) – Multi-container orchestration.
- [dive](https://github.com/wagoodman/dive) – Analyze and optimize Docker image layers.
- [docui](https://github.com/skanehira/docui) – Terminal UI for managing Docker.
- [dockly](https://github.com/lirantal/dockly) – Interactive terminal-based Docker management.

## GUI Tools

- [Portainer](https://github.com/portainer/portainer) – Web UI for managing Docker environments.
- [LazyDocker](https://github.com/jesseduffield/lazydocker) – Terminal UI for container management.
- [Rancher Desktop](https://github.com/rancher-sandbox/rancher-desktop) – Alternative to Docker Desktop.
- [Kitematic (Archived)](https://github.com/docker/kitematic) – Early GUI for Docker containers.

## Images & Base Images

- [Alpine Linux](https://hub.docker.com/_/alpine) – Tiny base image for minimal containers.
- [Ubuntu](https://hub.docker.com/_/ubuntu) – Common base image for full Linux distributions.
- [Distroless Images](https://github.com/GoogleContainerTools/distroless) – Minimalist images containing only application dependencies.
- [Nginx](https://hub.docker.com/_/nginx) – Popular web server container.
- [Node.js](https://hub.docker.com/_/node) – Official Node container images.
- [Python](https://hub.docker.com/_/python) – Python official base images.
- [Postgres](https://hub.docker.com/_/postgres) – PostgreSQL DB images.

## Docker Compose

- [Docker Compose](https://github.com/docker/compose) – Official multi-container orchestrator.
- [Awesome Compose Examples](https://github.com/docker/awesome-compose) – Official collection of real-world compose setups.
- [Podman Compose](https://github.com/containers/podman-compose) – Compose for Podman environments.

## Development Tools

- [Dev Containers](https://github.com/devcontainers/spec) – Standard for reproducible dev environments.
- [Tilt](https://github.com/tilt-dev/tilt) – Live-updating dev environments for microservices.
- [Okteto](https://github.com/okteto/okteto) – Cloud dev environments for containerized apps.
- [skaffold](https://github.com/GoogleContainerTools/skaffold) – CI/CD and dev automation for Kubernetes.

## Security

- [Trivy](https://github.com/aquasecurity/trivy) – Vulnerability and config scanner for containers.
- [Grype](https://github.com/anchore/grype) – Vulnerability scanner for container images.
- [Dockle](https://github.com/goodwithtech/dockle) – Dockerfile linter focused on security.
- [Hadolint](https://github.com/hadolint/hadolint) – Dockerfile linter and best practices checker.
- [Snyk Container](https://github.com/snyk/snyk) – Security scanning for images.

## Monitoring & Logging

- [cAdvisor](https://github.com/google/cadvisor) – Monitor container performance and resource usage.
- [Prometheus + Node Exporter](https://github.com/prometheus/prometheus) – Metrics collection and alerting.
- [Grafana](https://github.com/grafana/grafana) – Dashboards for container monitoring.
- [Loki](https://github.com/grafana/loki) – Log aggregation for container workloads.
- [Fluent Bit](https://github.com/fluent/fluent-bit) – Lightweight log processor for containers.
